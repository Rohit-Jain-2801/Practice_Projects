Tokenizer¶

This class allows to vectorize a text corpus, by turning each text into either a sequence of integers or into a vector where the coefficient for each token could be binary, based on word count, based on tf-idf...

  • num_words: the maximum number of words to keep, based on word frequency. Only the most common num_words-1 words will be kept.
  • filters: characters to be ignored. Default is all punctuations and line breaks and tabs.
  • lower: if True, texts are converted to lowercase
  • split: separator for word splitting
  • char_level: if True, every character will be treated as a token (basically character-encoding).
  • oov_token: replacing out-of-vocabulary words during text_to_sequence calls and should be unique

0 is a reserved index that won't be assigned to any word.

Padding¶

Pads sequences to the same length.

  • maxlen: if not provided, it is the length of the longest sequence in the list
  • padding: sequences shorter than maxlen are padded on either side (pre or post)
  • truncating: sequences longer than maxlen are truncated on either side (pre or post)
